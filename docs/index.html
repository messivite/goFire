GoFire

Firebase kimlik doğrulamalı API'ler oluşturmak için Go toolkit'i. Kod üretimi ve Vercel deployment desteğiyle.

Özellikler

  • Firebase Auth middleware – Bearer token doğrulama (dosya yolu veya JSON credentials)
  • api.yaml – Endpoint'leri YAML ile tanımlayın, Go handler ve route'ları otomatik üretin
  • CLI – gofire add, gofire gen, gofire list, gofire deploy
  • Vercel-ready – Tek komutla serverless deployment
  • Chi router – Hafif, Go HTTP routing

Hızlı Başlangıç

# CLI'yı yükle
go install github.com/messivite/goFire/cmd/gofire@latest

# Yeni proje oluştur
gofire init

# Yapılandır (port, Firebase, Redis)
gofire setup

# Endpoint ekle
gofire add endpoint "GET /users"
gofire add endpoint "POST /users --auth"
gofire add endpoint "GET /users/:id --auth"

# Handler ve server üret
gofire gen

# Yerelde çalıştır
go run ./cmd/server
