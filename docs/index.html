GoFire

Firebase ile kimlik doğrulamalı API'ler oluşturmak için Go toolkit'i. YAML ile endpoint tanımla, handler'ları otomatik üret, tek komutla Vercel'e deploy et.

Kurulum CLI'yı yükle ve ilk projeyi oluştur api.yaml Endpoint'leri YAML ile tanımla Firebase Auth Bearer token ile koruma Deploy Vercel'e tek komutla yükle

Özellikler

  • Firebase Auth middleware – Bearer token doğrulama (dosya veya JSON credentials)
  • api.yaml – Endpoint'leri YAML ile tanımla, Go handler ve route'lar otomatik üretilir
  • CLI – gofire add, gofire gen, gofire list, gofire deploy
  • Vercel-ready – Tek komutla serverless deployment
  • Chi router – Hafif, idiomatik Go HTTP routing

Hızlı Başlangıç

# CLI'yı yükle
go install github.com/messivite/goFire/cmd/gofire@latest

# Yeni proje oluştur
gofire init

# Yapılandır (port, Firebase, Redis)
gofire setup

# Endpoint ekle
gofire add endpoint "GET /users"
gofire add endpoint "POST /users --auth"

# Kod üret ve çalıştır
gofire gen
go run ./cmd/server
