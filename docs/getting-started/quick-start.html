Hızlı Başlangıç

5 adımda ilk GoFire API'nizi oluşturun.

1. Proje oluştur

mkdir my-api && cd my-api
go mod init my-api
go get github.com/messivite/goFire
gofire init

go mod init ile Go modülü oluşturulur. go get ile GoFire bağımlılığı eklenir. gofire init varsayılan api.yaml, cmd/server/main.go ve health endpoint'lerini oluşturur.

Önemli: go.mod olmadan gofire gen çalışmaz. Bu adımları atlamayın.

2. Yapılandır

gofire setup

Port, Firebase credentials ve Redis ayarlarını interaktif olarak yapılandırın. Değerleri .env dosyasına kaydedebilirsiniz.

3. Endpoint ekle

gofire add endpoint "GET /users"
gofire add endpoint "POST /users --auth"
gofire add endpoint "GET /users/:id --auth"

--auth parametresi endpoint'i Firebase Auth ile korur.

4. Kod üret

gofire gen
go mod tidy

Handler stub'ları ve server route'larını api.yaml'dan üretir. go mod tidy bağımlılıkları indirir ve go.sum oluşturur.

5. Sunucuyu çalıştır

gofire init zaten cmd/server/main.go oluşturmuştur. gofire gen sonrası doğrudan çalıştırın:

go run ./cmd/server

Sunucu varsayılan olarak http://localhost:8080 adresinde çalışır. Proje kök dizininden go run ./cmd/server çalıştırın (terminalde my-api/ içinde olun).

missing go.sum / not a main package? go mod tidy çalıştırın. go run . yerine go run ./cmd/server kullanın.
