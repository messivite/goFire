Firebase Auth

Firebase Auth opsiyoneldir. Aktif etmek için:

1. Service account indir

Firebase Console'dan service account JSON dosyasını indirin.

2. Ortam değişkenini ayarla

FIREBASE_CREDENTIALS_PATH=./service-account.json

3. api.yaml'da auth kullan

auth: true olan endpoint'ler Authorization: Bearer <token> header'ı gerektirir.

Vercel'de

Vercel'de dosya yolu yerine JSON string kullanın:

FIREBASE_CREDENTIALS_JSON='{"type":"service_account",...}'

Veya:

vercel env add FIREBASE_CREDENTIALS_JSON production
