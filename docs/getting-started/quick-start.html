Hızlı Başlangıç

1. Proje oluştur

mkdir my-api && cd my-api
go mod init my-api
gofire init

gofire init varsayılan api.yaml ve health endpoint'lerini oluşturur.

2. Yapılandır

gofire setup

Port, Firebase credentials ve Redis ayarlarını interaktif olarak yapılandırın. Değerleri .env dosyasına kaydedebilirsiniz.

3. Endpoint ekle

gofire add endpoint "GET /users"
gofire add endpoint "POST /users --auth"
gofire add endpoint "GET /users/:id --auth"

--auth parametresi endpoint'i Firebase Auth ile korur.

4. Kod üret

gofire gen

Handler stub'ları ve server route'larını api.yaml'dan üretir.

5. Çalıştır

go run ./cmd/server

Sunucu varsayılan olarak http://localhost:8080 adresinde çalışır.
