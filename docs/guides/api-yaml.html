api.yaml

Tüm endpoint'ler api.yaml dosyasında tanımlanır. Bu dosyayı doğrudan düzenleyebilir veya CLI ile endpoint ekleyebilirsiniz.

Örnek api.yaml

version: "1"
basePath: /api

endpoints:
  - method: GET
    path: /api
    handler: Health
    auth: false
  - method: GET
    path: /api/health
    handler: Health
    auth: false
  - method: GET
    path: /users
    handler: ListUsers
    auth: true
  - method: POST
    path: /users
    handler: CreateUsers
    auth: true
  - method: GET
    path: /users/:id
    handler: GetUsersById
    auth: true

Alanlar

  • auth: true – Route Firebase Auth middleware ile korunur
  • auth: false – Route herkese açıktır
  • handler – handlers/ dizininde üretilecek fonksiyon adı
