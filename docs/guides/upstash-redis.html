Upstash Redis

gofire setup sırasında Redis cache'i etkinleştirirseniz, cfg.RedisEnabled true olur ve cache paketini kullanabilirsiniz.

Kullanım

import "github.com/messivite/goFire/cache"

if cfg.RedisEnabled {
    c, _ := cache.NewUpstashCache(cfg.UpstashRedisRestURL, cfg.UpstashRedisRestToken, "myapp:")
    // Handler veya servislere inject edin
}

// Handler'da: c.Get(ctx, key), c.SetAsync(ctx, key, data, ttlSeconds)

Ortam değişkenleri

  • UPSTASH_REDIS_REST_URL – örn. https://your-db.upstash.io
  • UPSTASH_REDIS_REST_TOKEN
