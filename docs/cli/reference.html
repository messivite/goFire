CLI Komutları

KomutAçıklama
gofire initVarsayılan api.yaml ve health endpoint'lerini oluşturur
gofire setupİnteraktif yapılandırma (port, Firebase, Redis) ve .env'e kaydetme
gofire add endpoint "METHOD /path" [--auth]api.yaml'a endpoint ekler
gofire genapi.yaml'dan handler stub'ları ve server route'larını üretir
gofire listapi.yaml'daki tüm endpoint'leri listeler
gofire deployİnteraktif Vercel deploy (preview veya production)

Örnekler

gofire add endpoint "GET /users"
gofire add endpoint "POST /users --auth"
gofire add endpoint "GET /users/:id --auth"
