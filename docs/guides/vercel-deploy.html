Vercel Deployment

CLI ile deploy

gofire deploy

İnteraktif olarak preview veya production deploy seçebilirsiniz.

Manuel deploy

vercel --prod

Ortam değişkenleri

Firebase credentials gibi değişkenleri Vercel'e ekleyin:

vercel env add FIREBASE_CREDENTIALS_JSON production

Vercel'de FIREBASE_CREDENTIALS_PATH yerine FIREBASE_CREDENTIALS_JSON kullanın (tam JSON string).
