Go modülü olarak CLI olarak Gereksinimler

Kurulum

GoFire'ı Go modülü olarak kullanabilir veya CLI aracını global olarak yükleyebilirsiniz.

Go modülü olarak

go get github.com/messivite/goFire

CLI olarak

GoFire CLI'yı yüklemek için:

go install github.com/messivite/goFire/cmd/gofire@latest

Kurulumdan sonra gofire komutunu terminalde kullanabilirsiniz. PATH içinde $GOPATH/bin veya $HOME/go/bin olmalıdır.

Gereksinimler

  • Go 1.19+
  • Firebase projesi (opsiyonel, auth için)
  • Upstash Redis hesabı (opsiyonel, cache için)
